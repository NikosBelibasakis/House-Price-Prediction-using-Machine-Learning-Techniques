# House Price Prediction using Machine Learning

This project was developed as part of the IBM Generative AI Engineering Professional Certificate and focuses on predicting residential property prices using Machine Learning techniques. The workflow includes data preprocessing, exploratory data analysis (EDA), feature engineering, model development, and evaluation using Linear, Polynomial, and Ridge Regression models.

The repository contains both the final project report and the Jupyter Notebook implementation, allowing readers to review the methodology, results, and source code used throughout the project.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Repository Contents
- `House Price Prediction using Machine Learning Techniques.pdf` – Final project report
- `House_Sales_in_King_Count_USA.ipynb` – Jupyter Notebook containing the complete implementation and code
